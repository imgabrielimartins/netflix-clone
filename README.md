# Netflix Clone 🎬

Aplicação desenvolvida como um clone da interface da Netflix, utilizando apenas HTML, CSS e JavaScript.

O projeto simula a tela de seleção de perfis da plataforma, com foco em praticar conceitos de estruturação de páginas, estilização, responsividade e interatividade.

## Preview

A aplicação apresenta:

* Tela inicial inspirada na Netflix
* Seleção de perfis
* Ícones personalizados para cada usuário
* Alternância entre tema claro e escuro
* Layout centralizado e responsivo
* Interface moderna com foco na experiência visual

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## Estrutura do Projeto

```bash
├── index.html
├── style.css
├── theme.js
├── assets
```

## Funcionalidades

* Exibição de múltiplos perfis
* Efeito hover nos perfis
* Mudança de tema através do botão no canto superior direito
* Navegação simples e intuitiva

## Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de praticar:

* Estruturação de páginas com HTML
* Estilização com CSS
* Manipulação do DOM com JavaScript
* Organização de componentes visuais
* Criação de interfaces inspiradas em aplicações reais

## Como Executar o Projeto

1. Clone este repositório:

```bash
git clone https://github.com/imgabrielimartins/netflix-clone.git
```

2. Acesse a pasta do projeto:

```bash
cd netflix-clone
```

3. Abra o arquivo `index.html` no navegador.
